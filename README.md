## âš™ï¸ Configuration

### `accounts.json`

```json
[
  {
  {
    "cookie": "s%3Axxxxxxxxxx.yyyyyyyyyyyy",
    "proxy": "",
    "identifier": "akun1_noproxy"
  },
  {
    "cookie": "s%3Axxxxxxxxxx.yyyyyyyyyyyy",
    "proxy": "http://user:pass@proxy1.com:8080",
    "identifier": "akun2_single_proxy"
  }
]
```

| Field | Description |
|-------|-------------|
| `cookie` | Your `connect.sid` cookie value from the DeltaHash website |
| `proxy` | (Optional) HTTP or SOCKS5 proxy URL. Leave empty `""` for direct connection |
| `identifier` | Unique identifier for this account (used for fingerprint generation) |

### How to get your `connect.sid` cookie:

1. Login to https://portal.deltahash.ai
2. Open DevTools (F12) â†’ Application â†’ Cookies
3. Copy the value of `connect.sid`
4. Paste it in `accounts.json`
